# SBToWxFileUpload

## 简介
基于SpringBoot通用框架下的针对微信小程序的图片（文件）上传，Demo实现，附带小程序页面源码

## 特点
便捷，快速，api

## 配置

    application-dev.yml 数据库配置
    com.myself.wechatfileupload.util 服务器文件存储地址
